- [ ] **Step 5: Survey the three options for resolving drift**

For each option below, write pros / cons / open questions:

a) **Gitignore `web/content/`, regenerate at build**
   - Sync runs in Vercel build via `pnpm build` (would need a `prebuild` hook calling `cargo run --bin sync_content`)
